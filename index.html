Projects

Hierarchical Probabilistic Vote Regression on Galaxy Zoo 2 using Vision Transformers

Models galaxy morphology on the Galaxy Zoo 2 dataset as probabilistic regression rather than hard-label classification. The model predicts full vote distributions instead of single labels, preserving the uncertainty and disagreement present in human annotations.

Vision Transformers Probabilistic Regression Galaxy Zoo 2

DSR using BiLSTM-GRU

Recognizes dysarthric speech patterns using a BiLSTM-GRU model, benchmarked against a Hidden Markov Model baseline. Trained on digit recordings from the UASpeech dataset (University of Illinois) — 770 files across all digit classes.

BiLSTM-GRU HMM Baseline UASpeech

Paddy Disease Classification using ML and DL Techniques

Classifies paddy leaf diseases — bacterial leaf blight, brown spot, blast, tungro, hispa, and more — comparing classical machine learning against deep learning approaches. Each disease category trained on 450 image samples.

Deep Learning Classical ML Image Classification
